#  Week 3 – SOC Operations & Incident Handling

This repository contains practical implementations of Security Operations Center (SOC) tasks including log analysis, threat intelligence integration, incident escalation, alert triage, evidence collection, and full workflow simulation.

---

## 📂 Tasks Overview

### 🔹 Task 01 – Advanced Log Analysis

Performed log correlation, anomaly detection, and log enrichment using SIEM tools. Identified suspicious patterns such as failed login attempts and abnormal network traffic.

---

### 🔹 Task 02 – Threat Intelligence Integration

Integrated AlienVault OTX with Wazuh to enrich alerts with threat intelligence. Validated indicators and performed threat hunting to identify suspicious activities.

---

### 🔹 Task 03 – Incident Escalation Practice

Simulated incident escalation using TheHive. Created cases, assigned tasks, added observables, and escalated incidents to Tier 2 with proper documentation.

---

### 🔹 Task 04 – Alert Triage with Threat Intelligence

Analyzed alerts in Wazuh and validated indicators using VirusTotal and OTX. Determined whether alerts were true positives or false positives.

---

### 🔹 Task 05 – Evidence Preservation & Analysis

Collected volatile data and memory dumps using Velociraptor. Maintained integrity of evidence using hashing techniques.

---

### 🔹 Task 06 – Capstone SOC Workflow Simulation

Simulated a complete SOC workflow including attack detection, triage, response, escalation, and reporting.
