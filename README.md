# Week 2 - SOC Practical Tasks

This repository contains practical SOC (Security Operations Center) tasks including alert management, incident response, evidence collection, and a complete capstone project.

---

## 📌 Task Overview

### 🔹 Task 1: Alert Management
- Created alert classification table
- Mapped alerts with MITRE ATT&CK techniques
- Assigned priorities and severity levels

---

### 🔹 Task 2: Incident Response Documentation
- Created incident response template
- Included timeline, impact analysis, remediation steps
- Prepared investigation logs and checklist

---

### 🔹 Task 3: Alert Triage Practice
- Analyzed alerts using Wazuh SIEM
- Identified false positives and true positives
- Documented alert triage process

---

### 🔹 Task 4: Evidence Preservation
- Used Velociraptor for evidence collection
- Collected system process and network data
- Maintained basic forensic workflow

---

### 🔹 Task 5: Capstone Project (Full Alert-to-Response Cycle)
- Simulated brute-force SSH attack
- Detected alerts using Wazuh
- Performed alert triage and analysis
- Executed response actions
- Collected forensic evidence using Velociraptor
